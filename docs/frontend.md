
# Frontend - DoQR Employee Register

## Estrutura
- **components:** Componentes reutilizáveis.
- **pages:** Rotas do Next.js.
- **services:** Integração com APIs.
- **utils:** Funções auxiliares.

## Como Rodar
1. Instale as dependências:
   ```bash
   cd doqr-employee-frontend
   npm install
   ```
2. Rode o frontend:
   ```bash
   npm run dev
   ```
